# Accountant-App

This Proyect is a transaction recorder Aplication using React.JS

### Functionalities:
1. Show income.
2. Show Expenses.
3. Show Total.
4. Keep a record of Transactions. 

### Technologies used:
1. ReactJs.
2. LocalStorage.
3. Netlify.
